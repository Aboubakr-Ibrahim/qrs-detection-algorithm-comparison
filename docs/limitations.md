# Limitations and responsible use

- These are research implementations, not validated clinical detectors.
- Results depend on record selection, channel, annotation policy, tolerance, noise, pacing, and arrhythmia composition.
- The Pan–Tompkins implementation follows the algorithmic style but is not claimed as a bit-for-bit reproduction of every historical implementation.
- Wavelet results depend on family, level, reconstruction choice, and threshold.
- Aggregate metrics must be calculated from pooled TP/FP/FN counts or reported alongside record-level distributions.
- Predecessor performance numbers are not claimed until reproduced using this repository.
- No patient data is included. Do not use the code for diagnosis, monitoring, alarms, or treatment.
